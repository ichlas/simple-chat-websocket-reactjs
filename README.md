## Development

```bash
# install dependencies
$ git clone https://github.com/ichlas/simple-chat-websocket-reactjs.git

# serve with hot reload at localhost:3000
$ yarn start_client

# build for production and launch server
$ yarn start_server

# generate static project
$ yarn generate
```
